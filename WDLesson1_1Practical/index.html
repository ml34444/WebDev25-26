Web Technologies

HTML

HTML is one of the three languages needed in order to create webpages. HTML is responsible for adding elements to a webpage. Examples of elements that HTML can import onto a webpage are text and images.

CSS

CSS is one of the three coding languages used to create webpages. CSS is responsible for adding detail towards web pages. For example, CSS would take the code from HTML and style it by adding color and changing font sizes to make it more visually appealing.

JavaScript

Javascript is the third language used to create webpages. Javascript is responsible for making a webpage more interactable. Examples of this would be increasing the size of a button whether the mouse is hovering over it or creating side menus for users to interact with.
